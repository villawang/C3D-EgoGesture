# C3D-EgoGesture
C3D model for EgoGesture video recognition

torch == 1.2 \
torchvision == 0.4.0 \
scipy \
sklearn \
tqdm \
torchsummary \
pillow==6.2.1 \
matplotlib \
opencv-python-headless \
pandas \
scikit-image

First use create_annotation.py to create annotation files.
